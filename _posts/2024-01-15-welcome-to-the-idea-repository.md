## Welcome!

This is a collaborative knowledge library for mathematical theories, study notes, and original ideas. The site is built with [Jekyll](https://jekyllrb.com/) and hosted on [GitHub Pages](https://pages.github.com/), making it completely free and open-source.

## Features

- **MathJax Support**: Write mathematical equations using LaTeX syntax
- **Version Control**: All content is stored in Git for full history tracking
- **Community Driven**: Anyone can contribute via pull requests
- **No Backend Required**: Static site with zero server maintenance

## Example Equations

### Inline Equations

Euler's identity: $e^{i\pi} + 1 = 0$

### Display Equations

The quadratic formula:
$$
x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}
$$

### Complex Equations

The Gaussian integral:
$$
\int_{-\infty}^{\infty} e^{-x^2} dx = \sqrt{\pi}
$$

## How to Contribute

1. Fork the repository
2. Copy the [submission template](ideas/_submissions/template.md)
3. Create a new file in `_posts/` with your content
4. Submit a pull request

See [CONTRIBUTING.md](CONTRIBUTING.md) for detailed instructions.
